# Локальная админка финансов

## Описание

Этот репозиторий — эксперимент по созданию собственной **финансовой админки нового поколения**. Задача окружения — собрать воедино понятный, лёгкий и красивый инструмент для анализа личных и бизнес-финансов.

Проект задумывается как **локальное одностраничное приложение**, работающее без внешнего бэкенда. Все данные, правила и настройки хранятся в браузере или локальных файлах пользователя. Это позволяет сохранить приватность, ускорить работу и упростить запуск — достаточно скачать проект и открыть в браузере.

Основные цели окружения:

* **Импорт CSV** из банков (Т-Банк, Альфа-Банк и др.) с автоматическим определением кодировки и разделителя.
* **Дедупликация операций** по `external_id` или «фингерпринту», чтобы не плодить дубликаты.
* **Исключение «Инвесткопилки»** и других шумных операций, чтобы аналитика оставалась чистой.
* **Нормализация контрагентов** и распределение по категориям (Еда, Транспорт, Отношения, Подписки, Жильё и др.), включая ручную донастройку.
* **Гибкий словарь правил** (по ключевым словам или MCC), который помогает автоматически подхватывать новые операции.
* **Интерактивные отчёты и дашборды**:
  * P\&L по месяцам (доходы, расходы, чистый поток),
  * структура расходов (pie),
  * динамика по категориям и кумулятивный поток,
  * топ контрагентов и активность по дням недели.
* **Современный UI/UX**: тёмная тема, чистая визуализация, быстрые фильтры и понятные пустые состояния («Нет данных — загрузите CSV»).

Почему это окружение важно:

* Большинство доступных сервисов либо перегружены лишними функциями, либо привязывают к облаку. Здесь же акцент на простоте, прозрачности и контроле над своими данными.
* Всё строится вокруг **категорий и визуализации**: данные должны не пугать, а помогать принимать решения.
* В будущем планируется расширение: регулярные расходы, аномалии, сравнение план/факт.

Технический стек: **React + Vite + TypeScript + Recharts**. Хранение — LocalStorage. Никакого API, никаких внешних сервисов — всё работает локально.

Итоговое окружение — это не просто табличка, а **понятная и красивая система управления личными и бизнес-финансами**, с которой можно ежедневно работать и постепенно строить собственную аналитику.

## Возможности

- Поддержка крупных CSV (5–10 тысяч строк) с автоматическим определением кодировки, разделителя и параллельным парсингом.
- Дедупликация операций по `external_id` или хэш-фингерпринту.
- Распознавание «Инвесткопилки» и глобальный тумблер включения/исключения.
- Нормализация контрагентов, словарь категорий и базовый набор правил по ключевым словам.
- Ручная категоризация и создание правил по контрагентам с живым пересчётом.
- Дашборд с KPI, P&L по месяцам, кумулятивным потоком, структурой расходов, динамикой категорий, активностью по дням недели, топами контрагентов и распределением чеков.
- Разделы «Операции», «Контрагенты», «Правила», «Инвесткопилка», «Регулярные», «Аномалии», «Экспорт» с фильтрами и виртуальным списком для работы с большими данными.
- Экспорт текущего среза и словаря правил в локальные файлы.
- Сохранение состояния (операции, фильтры, правила) в LocalStorage.

## Разработка

```bash
npm install
npm run dev
```

Сборка и проверки:

```bash
npm run build
npm run lint
```

Приложение запускается локально на `http://localhost:5173`.
