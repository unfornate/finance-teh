# Локальная админка финансов

Одностраничное приложение для загрузки банковских CSV и оперативного анализа личных финансов. Все данные и правила хранятся в браузере, внешний бэкенд не используется.

## Возможности

- Поддержка нескольких CSV с автоматическим определением кодировки и разделителя.
- Дедупликация операций по `external_id` или хэш-фингерпринту.
- Распознавание «Инвесткопилки» и глобальный тумблер включения/исключения.
- Нормализация контрагентов, словарь категорий и базовый набор правил по ключевым словам.
- Ручная категоризация и создание правил по контрагентам с живым пересчётом.
- Дашборд с KPI, P&L по месяцам, структурой расходов и топами контрагентов.
- Разделы «Операции», «Контрагенты», «Правила», «Инвесткопилка», «Регулярные», «Аномалии», «Экспорт».
- Экспорт текущего среза и словаря правил в локальные файлы.
- Сохранение состояния (операции, фильтры, правила) в LocalStorage.

## Разработка

```bash
npm install
npm run dev
```

Сборка и проверки:

```bash
npm run build
npm run lint
```

Приложение запускается локально на `http://localhost:5173`.
